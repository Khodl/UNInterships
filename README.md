# UN Internships

Browse the latest internship and job opportunities across United Nations agencies and international organizations. Powered by [UNtalent.org](https://untalent.org).

Built with [Nuxt 3](https://nuxt.com) and Tailwind CSS.

## Setup

```bash
npm install
```

Create a `.env` file:

```
NUXT_UNTALENT_TOKEN=your_untalent_api_token
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
